**`name`** u8aConcat

**`summary`** Creates a concatenated Uint8Array from the inputs.

**`description`** 
Concatenates the input arrays into a single `UInt8Array`.

**`example`** 
<BR>

```javascript
import { u8aConcat } from '@polkadot/util';

u8aConcat(
  new Uint8Array([1, 2, 3]),
  new Uint8Array([4, 5, 6])
); // [1, 2, 3, 4, 5, 6]
```
